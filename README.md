# Random Password Generator

This application generates random passwords based on user preferences including numbers, alphabets, and special characters.

## Assumptions
- Password length is fixed at 12 characters.
- Only the last 5 passwords are stored and displayed.

## How to Use
- Check the options for including numbers, alphabets, and special characters.
- Click "Generate Password" to create a new password.
- Click "Copy to Clipboard" to copy the password.

## Deployment
- The application is deployed on Vercel: https://password-generator-tools.vercel.app/

## Local Development
- Clone the repository: `git clone https://github.com/KhushalKoriya/PasswordGenerator.git`
- Install dependencies: `npm install`
- Start the development server: `npm start`
